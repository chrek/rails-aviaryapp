# rails-aviaryapp

Aviary App with Posts based on Rails
------------------------------------

This site is using Rails.

## Database: Sqlite3

## What has been implemented

* Navigation
* Nested Resources
* Use of layouts and include files 

To test the site locally:

From the root of project run the commands:

1. `bundle install`
2. `rails serve`
